# **AVGO CLIENT SIDE**
**Getting Started with AVGO Client Side:**
**Note: Before starting make sure you have node(19.1.0) and npm(8.19.3) installed.**
- Step 1: Clone repo
`git clone https://github.com/chwahajahmad/fyp-client`
- Step 2: Open repo
`cd fyp-client`
- Step3: Install dependencies:
`npm i`
- Step4: Run Program
`npm run start`
